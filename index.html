🔌 Localisation port OBD2 par marque
ℹ️Le port OBD2 (EOBD) est obligatoire sur tous les véhicules essence depuis 2001 et diesel depuis 2003 en Europe. Il est généralement sous le tableau de bord côté conducteur.

🇯🇵 Toyota / Lexus

Sous le tableau de bord gauche, près de la colonne de direction. Modèles récents: facile d'accès côté conducteur.

🇯🇵 Honda / Acura

Certains modèles: côté passager sous tableau de bord. La plupart: côté conducteur. Prévoir une lampe de poche.

🇯🇵 Nissan / Infiniti

Sous le tableau de bord gauche, souvent derrière un petit volet plastique. Visible sans démontage.

🇯🇵 Mazda

Côté conducteur sous le tableau de bord. Accessible directement, parfois protégé par couvercle.

🇯🇵 Subaru / Mitsubishi

Sous le tableau de bord côté conducteur, vers le centre. Peut nécessiter de se pencher légèrement.

🇯🇵 Suzuki / Daihatsu

Sous le tableau de bord gauche. Généralement sans couvercle, visible directement.

🇩🇪 BMW / Mini

Sous le tableau de bord gauche, à gauche de la colonne. Souvent dans compartiment dédié avec couvercle.

🇩🇪 Mercedes-Benz

Sous le tableau de bord à gauche. Certains modèles ont un couvercle arborant le logo MB.

🇩🇪 Volkswagen / Audi / Skoda / Seat

Sous le tableau de bord côté conducteur, à gauche. Souvent derrière une petite trappe plastique.

🇫🇷 Peugeot / Citroën

Sous la planche de bord côté conducteur. Parfois nécessite de regarder en bas à gauche.

🇫🇷 Renault

Sous le tableau de bord à gauche, généralement sans couvercle. Accès facile sur la majorité des modèles.

🇸🇪 Volvo

Sous le tableau de bord gauche. Modèles anciens: près du cendrier. Modèles récents: visible facilement.

🇺🇸 Ford / Lincoln

Sous le tableau de bord côté conducteur. Camions: parfois sous le tableau de bord central.

🇺🇸 Chevrolet / GMC / Cadillac

Sous le tableau de bord à gauche. Sur camionnettes: sous le tableau de bord central ou colonne direction.

🇺🇸 Dodge / Jeep / Chrysler

Sous le tableau de bord côté conducteur, généralement visible sans outil.

🇰🇷 Hyundai / Kia

Sous le tableau de bord gauche, près du pédalier. Accessible directement sans outils.

Procédure type — Codage module
  1. Connecter l'interface VCDS (USB) → moteur coupé, clé sur contact
  2. Ouvrir VCDS → Select Control Module → choisir le module (ex: 01 – Engine)
  3. Fault Codes – 02 pour lire, Clear Codes – 05 pour effacer
  4. Measuring Blocks – 08 → entrer les numéros de blocs pertinents
  5. Pour coder : Coding – 07 → modifier la valeur (noter l'ancienne valeur avant !)
  6. Pour adapter : Adaptation – 10 → sélectionner le canal → modifier la valeur
  7. Pour tester actuateurs : Output Tests – 03 → suivre la séquence
  8. Consulter ross-tech.com/vcds pour les valeurs exactes par véhicule
Toujours noter les valeurs originales avant tout codage. Un mauvais codage peut désactiver un système de sécurité ou empêcher le démarrage.

Marques couvertes
Toyota (toutes)Lexus Daihatsu (partiel)
Procédure type — Diagnostic hybride
  1. Connecter le câble Mini-VCI (USB) sur port OBD2 — véhicule en mode READY
  2. Ouvrir Techstream → le VIN est détecté automatiquement
  3. Powertrain → HV Battery System → Trouble Codes
  4. Lire les codes P3xxx et P0Axx → noter le freeze frame (SOC, tension HV, T° cellules)
  5. Data List → sélectionner : HV Battery Voltage, HV Battery Current, Battery Block Voltage 1–16
  6. Identifier les blocs de cellules avec tension déviante (écart > 0,3V suspect)
  7. Si remplacement batterie : Utility → HV Battery System → Battery Reset
  8. Après réparation : effacer codes, faire 3 cycles charge/décharge pour calibrer le BMS
🚨 Ne jamais travailler sur le circuit HV sans EPI (gants isolants 1000V, tapis isolant). Couper le service plug AVANT toute intervention physique.

Procédure type — Codage batterie neuve
  1. Connecter interface ENET (câble RJ45 ↔ OBD2) ou ICOM Next
  2. Ouvrir ISTA → sélectionner Vehicle Identification → laisser le VIN s'auto-remplir
  3. Aller dans Service Functions → Power Management → Battery Registration
  4. Entrer la capacité de la nouvelle batterie (Ah), le type (AGM/EFB/Gel) et la référence
  5. Valider → ISTA recalibre le BMS pour la nouvelle batterie

Ne jamais interrompre une session de programmation (flash). Une coupure courant pendant le flash peut rendre le module inutilisable (brick). Utiliser une alimentation stabilisée 14,4V.

Procédure type — Modification As-Built

⚙️ Procédure générale de diagnostic
  1. Brancher le scanner OBD2 sur le port DLC (moteur allumé ou clé sur position MAR/ON)
  2. Lire TOUS les codes défauts présents — noter les codes actifs ET en mémoire (pendants)
  3. Photographier les données du freeze frame (conditions au moment du défaut: régime, charge, température)
  4. Rechercher chaque code dans cette base pour identifier les causes probables et probabilités
  5. Vérifier d'abord les causes simples : niveaux, câblage, connecteurs, fusibles
  6. Effectuer les tests de diagnostic dans l'ordre (du plus simple au plus complexe)
  7. Réparer ou remplacer les composants défectueux identifiés
  8. Effacer les codes défauts après réparation
  9. Faire un test routier de 20–30 km incluant les conditions qui ont déclenché le défaut
  10. Relire les codes pour confirmer que le défaut ne revient pas (cycle drive complet)
